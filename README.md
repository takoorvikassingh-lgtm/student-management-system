# Student Management System

## Description
The Student Management System is a beginner-friendly Python project that helps manage student records using a menu-driven program. It allows users to add, view, search, and delete student details.

## Features
- Add Student
- View Students
- Search Student
- Delete Student
- Exit Program

## Technologies Used
- Python

## Concepts Used
- Variables
- Lists
- Dictionaries
- Loops
- Conditional Statements
- User Input

## How to Run

1. Install Python.
2. Download or clone this project.
3. Open the project in VS Code.
4. Run the program using:

```bash
python student_management.py
```

## Sample Menu

```
===== Student Management System =====

1. Add Student
2. View Students
3. Search Student
4. Delete Student
5. Exit
```
